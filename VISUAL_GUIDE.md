## Interactive Elements

```
Buttons:
- Gradient background
- Rounded corners
- Hover: Elevation + shadow
- Active: Scale down slightly

Form Inputs:
- Border: 2px solid light gray
- Focus: Purple border + glow
- Placeholder: Light gray

Cards:
- Shadow on normal
- Larger shadow on hover
- Transform: translate up on hover
```
